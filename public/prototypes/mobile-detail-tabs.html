What this prototype addresses

Problem
Detail modal on mobile is a scrolling wall of content — body, activity, and thread all compete in one pane
Approach
Full-screen takeover with 4-tab nav: Overview (meta + status), Body (content + AC), Activity (log), Discussion (thread)
What's real
Detail modal at features/vault-item/; 3-column desktop layout; activity + thread services
New: tab strip collapses 3-column layout into 4 tabs; sticky header + action row always visible
